# stim2KO_single_cells
This project utilized probabilistic variational inference methods and manifold learning-based methods to quantify the effects of *stim2* gene knockout (*stim2a*;*stim2b*<sup>-/-</sup> double KO)  at single cell resolution on nervous system development in zebrafish. The predictions from this analysis were experimentally validated and published in Baranykova et al, Sci Rep. 2024, PMID: 39424970 
https://pubmed.ncbi.nlm.nih.gov/39424970/ 

The code for this analysis is available upon request.
